# 🚗 AI Vehicle Service Management System

An AI-powered vehicle service booking and management platform that helps users schedule vehicle services, track maintenance history, monitor live service progress, and receive AI-based maintenance predictions.

---

# 📌 Project Overview

The AI Vehicle Service Management System is a smart platform designed to simplify vehicle servicing and maintenance management. Users can book services online, track repair progress in real-time, and receive intelligent maintenance recommendations using AI and machine learning.

The platform also includes dashboards for mechanics and administrators to manage bookings, customer data, analytics, and service records efficiently.

# ⚙️ Installation & Setup

## Clone Repository

```bash
git clone https://github.com/yourusername/ai-vehicle-service-system.git
```

## Navigate to Project Folder

```bash
cd ai-vehicle-service-system
```

## Install Frontend Dependencies

```bash
cd client
npm install
```

## Install Backend Dependencies

```bash
cd ../server
npm install
```

## Install Python Dependencies

```bash
pip install -r requirements.txt
```

---

# 🔑 Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
RAZORPAY_KEY=your_razorpay_key
CLOUDINARY_API_KEY=your_cloudinary_key
```

---

# 🚀 Run Application

## Start Backend

```bash
npm run server
```

## Start Frontend

```bash
npm run dev
```

## Start AI Model

```bash
python predict.py
```
